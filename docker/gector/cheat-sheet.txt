# Docker Cheat Sheet — GECToR Container

## Build the image (after any Dockerfile change)
docker build -t gector-env .

## Run the container with data + scripts mounted
docker run -it -v D:\Mestrado\docker\gector\data:/app/data -v D:\Mestrado\docker\gector\scripts:/app/scripts gector-env bash

## Run a script directly without entering the shell first
docker run -v D:\Mestrado\docker\gector\data:/app/data -v D:\Mestrado\docker\gector\scripts:/app/scripts gector-env python3 /app/scripts/gector_correct.py

## List built images
docker images

## List running containers
docker ps

## List all containers (including stopped)
docker ps -a

## Remove a stopped container by ID
docker rm <container_id>

## Remove an image
docker rmi gector-env

## Free up disk space from unused layers/images (run occasionally)
docker system prune
